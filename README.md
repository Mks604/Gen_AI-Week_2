Project: Simple Hugging Face Chatbot in Google Colab

How it works

The script loads a pretrained conversational model from Hugging Face.

You type your messages in the Colab input.

The model generates replies and maintains chat context.

Type "exit" to quit the chat.

✅ Example Interaction
🤖 Hugging Face Chatbot (type 'exit' to stop)

You: Hello

Bot: Hi! How are you today?

You: What is AI?

Bot: AI is artificial intelligence, which allows machines to learn and perform tasks.

You: exit

Bot: Goodbye!
